Omi Ntfs NTFS磁盘专家: Mac上好用、免费的读写 NTFS 磁盘工具

软件简介

软件简介Mac 上不能写 NTFS 格式的磁盘一直是很多用户感觉很不方便的地方。毕竟用 Windows 的人还是有很多，虽然自己的 U 盘和移动硬盘一般可以换成 exFAT 格式的，但别人一般都还是 NTFS 的。所以安装一款能读写 NTFS 的 Mac 软件可以说是很有必要的了能使 macOS 读写 NTFS 的工具有很多，但要么收费，而且很贵；要么像 NTFSTool 这种用爱发电的开源软件（作者很长时间没更新了，上一次发行还是2020年5月，BigSur 用不了），作者又随时可能不干了。所以这次找到一个免费、支持 BigSur、支持 M1 芯片的 NTFS 读写工具，特地来分享给大家。
