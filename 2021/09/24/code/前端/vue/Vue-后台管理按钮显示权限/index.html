Vue-后台管理按钮显示权限

前言

在后台项目中，不同的登录者有不同的角色，可能上面的编辑、删除、详情这些按钮不是每个人都能点击的。不同的人登录系统有不同的操作权限。

实现逻辑

  1. 通过点击左侧菜单栏目 或者 URL 获取到当前的路径 path

    如： https://domain.com/you-admin/index.html#/teacher 获取到信息为 path = '/teacher'

  2. 切换页面后从后台请求当前页面path 对应该用户有什么相对应的权限，进行存储。

  3. 在需要操作权限的位置进行 固定权限名称 进行比对 ,通过 v-if 进行控制显示和点击。
